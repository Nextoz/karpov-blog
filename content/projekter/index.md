## Karpov Blog

Personlig tech-blog bygget med [Hugo](https://gohugo.io/) og [PaperMod](https://github.com/adityatelange/hugo-PaperMod/), hostet på GitHub Pages. Dark theme, SEO-optimeret, dansk indhold om AI, DevOps og teknologipolitik.

**Stack:** Hugo, PaperMod, GitHub Actions, GitHub Pages

## SSOIDP — SSO Identity Provider

SSO Identity Provider for Arbejdstilsynet (Datatilsynet). IdentityServer8-baseret OIDC-provider med FIDO2/MFA, ASP.NET Identity og EF Core.

**Stack:** .NET 8, IdentityServer8, SQL Server, Azure DevOps, IIS

## DCAB — Det Centrale Almene Bygningsregister

Offentligt system til håndtering af bygningsdata i Landsbyggefonden. Blazor Server-applikation med REST APIs og Elasticsearch.

**Stack:** .NET 8, Blazor Server, MSSQL, Elasticsearch, Hangfire

## MyPaperFlow — Boyum IT Solutions

Cloud-baseret platform til dokumenthåndtering og SAP Business One-integration. DevOps og infrastruktur med Kubernetes og KEDA autoscaling.

**Stack:** Azure DevOps, Docker, Kubernetes, KEDA, Bicep, Terraform, PostgreSQL
